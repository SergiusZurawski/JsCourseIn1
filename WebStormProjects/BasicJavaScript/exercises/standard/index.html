JavaScript Exercises and Solutions

Exercises

Palindromes

Examine the file palindromes.html. Complete the function 'isAPalindrome' so it correctly determines if the input is a palindrome.

Object Orientation

Examine the file OO.html. Create a student type such that the code within will execute successfully.

A Functional Programming Challenge

Examine the file fp.html. Complete the function buildShapePrinter so that when called it returns a function that builds the specified type of shape with the specified dimension(s). Hint: wrap the output in a 'pre' element do avoid the browser styling your output.

Using the 'Functional Toolkit' Via Underscore.js

Examine the file usingUnderscore.html. Complete the code such that empty text boxes have the corresponding labels turned red and submission of the form is canceled (run the solution to see this working). Try to use the functional toolkit instead of your own loops and conditionals.

Writing Tests Using Jasmine

Examine the following files:

  • testingType.js
  • testingSpec.js
  • SpecRunnerBegin.html

These define a JavaScript type to represent a cell from the Game Of Life, a skeleton test to ensure that the type works correctly and a page to launch the test in Jasmine. Complete the test so that all behaviours are verified.

Extending the SCJP Quiz Part I

Examine the file quiz.html which contains sample questions for the SCJP exam. The first question contains an extra link inviting the user to view information about the correct answer. Attach an event handler to this link so that when it is clicked an extra paragraph is dynamically added at the bottom of the question. The default behavior of the link should be canceled.

Extending the SCJP Quiz Part II

Make a copy of your solution to the previous exercise and then extend it to work with the first four questions. Store the extra information to be added in a JSON-like data structure.

Extending the SCJP Quiz Part III

Make a copy of your solution to the previous exercise and then extend it so that the text for extra information is being retrieved from the server asynchronously. Assume we need to refresh the data every time extra info is requested.

Extending the SCJP Quiz Part IV

Create a form for booking a date and time to take the SCJP, similar to the one in the sample solution. Add code to validate the form on submission. Note that:

  1. The selection box must be populated via AJAX from 'data/testingCentres.json'
  2. The users candidate number should be checked against a regex.

Extending the SCJP Quiz Part V

Using the form developed in the previous exercise write code to:

  1. Preset the remaining elements of the form via 'Data/bookingPresets.json'
  2. Submit the form asynchronously by POST'ing the parameters to a Servlet
